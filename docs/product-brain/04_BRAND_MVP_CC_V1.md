# Cerveau Produit — Conseil constitutionnel — Lot 1

## Brand MVP

---

## 1. Rôle de la marque dans le projet

Le futur site du Conseil constitutionnel ne doit pas être traité comme une simple refonte graphique.

Son identité doit traduire une posture institutionnelle forte :

> **Rendre l’institution plus lisible, plus accessible et plus utile, sans affaiblir son autorité ni sa rigueur juridique.**

La marque d’interface doit soutenir trois objectifs :

1. inspirer confiance ;
2. faciliter la compréhension ;
3. donner envie d’approfondir.

Le site doit paraître :

- républicain ;
- moderne ;
- clair ;
- fiable ;
- sobre ;
- accessible ;
- institutionnel sans être froid ;
- expert sans être illisible.

---

## 2. Personnalité cible

La personnalité du futur site doit combiner plusieurs dimensions :

- clair ;
- pédagogique ;
- moderne ;
- sobre ;
- expert ;
- fiable ;
- accessible ;
- rassurant ;
- institutionnel sans être distant.

### Priorité donnée à

> **Clarté + Modernité + Fiabilité**

### Phrase de personnalité

> **Républicain dans l’autorité, moderne dans l’usage, clair dans la lecture, fiable dans la preuve.**

---

## 3. Posture institutionnelle

Le futur site doit incarner une institution qui reste solide, sérieuse et légitime, mais qui accepte de rendre ses contenus plus accessibles.

Il ne s’agit pas de rendre le Conseil constitutionnel “populaire” au sens marketing.

Il s’agit de rendre ses missions, ses décisions et ses contenus plus compréhensibles.

### Posture attendue

- autorité calme ;
- clarté démocratique ;
- sobriété visuelle ;
- pédagogie maîtrisée ;
- rigueur juridique ;
- accessibilité réelle ;
- modernité d’usage.

---

## 4. Direction artistique

La direction artistique doit être suffisamment républicaine pour incarner l’institution, mais suffisamment moderne pour éviter l’image d’un site administratif figé.

### Orientation recommandée

> **Républicain + moderne + média public**

Cela signifie :

- une identité sobre et institutionnelle ;
- une palette maîtrisée ;
- une hiérarchie typographique forte ;
- des espaces de respiration ;
- des composants clairs ;
- une mise en page éditoriale ;
- une valorisation des contenus complexes ;
- une esthétique sérieuse mais contemporaine.

La direction artistique ne doit pas chercher l’effet spectaculaire.

Elle doit produire une impression de :

- confiance ;
- lisibilité ;
- maîtrise ;
- sérieux ;
- ouverture.

---

## 5. Design system

Le design system attendu doit être compris comme un référentiel UX/UI, ergonomique et graphique.

Il doit formaliser :

- les principes d’interface ;
- les composants graphiques ;
- les règles d’usage ;
- les états interactifs ;
- les règles de hiérarchie visuelle ;
- les comportements responsive ;
- les principes d’accessibilité ;
- les bonnes pratiques de sobriété numérique.

Le design system n’a pas vocation à être livré comme un développement HTML/CSS/JS dans le cadre du lot 1.

Il doit être structuré dans un outil de conception adapté, comme Figma ou équivalent, afin de faciliter sa transmission au titulaire du lot 2.

---

## 6. Relation au DSFR

Le Conseil constitutionnel n’impose pas le recours exclusif au DSFR.

La proposition peut donc assumer une ligne graphique propre au Conseil constitutionnel, à condition de respecter :

- l’accessibilité ;
- l’ergonomie ;
- la cohérence ;
- la sobriété ;
- la maintenabilité ;
- l’image institutionnelle du Conseil.

Le design doit être justifié par rapport au contexte institutionnel et aux objectifs du projet.

---

## 7. Références d’intention

Le site peut s’inspirer de plusieurs familles d’interfaces.

### Interfaces institutionnelles

Pour :

- l’autorité ;
- la rigueur ;
- la stabilité ;
- la confiance.

### Médias publics

Pour :

- la clarté éditoriale ;
- la hiérarchie de l’information ;
- les formats d’explication ;
- l’accessibilité des contenus.

### Interfaces documentaires expertes

Pour :

- la recherche ;
- les références ;
- les filtres ;
- les métadonnées ;
- les citations ;
- les usages professionnels.

### Interfaces pédagogiques sobres

Pour :

- l’aide à la compréhension ;
- la structuration progressive ;
- l’explication sans infantilisation.

---

## 8. Ton éditorial

Le ton doit être :

- simple ;
- clair ;
- direct ;
- pédagogique mais sérieux ;
- expert quand nécessaire ;
- sobre ;
- sans marketing ;
- sans dramatisation ;
- sans effet de style inutile.

Le site doit expliquer sans surjouer.

Il doit rendre compréhensible sans réduire la complexité à l’excès.

### Ton recommandé

> **Clair, précis, sobre et fiable.**

---

## 10. Mots à éviter

Certains mots doivent être évités ou utilisés avec beaucoup de prudence.

### “Vulgarisation”

Le mot “vulgarisation” peut donner l’impression d’un abaissement du niveau de rigueur.

À privilégier :

- aide à la compréhension ;
- médiation ;
- lecture simplifiée ;
- repères de lecture ;
- explication progressive.

### “Réécriture”

Le mot “réécriture” est à éviter.

Il peut créer une confusion avec le texte officiel.

À privilégier :

- résumé éditorial ;
- aide à la lecture ;
- contenu d’explication ;
- mise en contexte ;
- repères de compréhension.

### “Simplification”

Le mot peut être utilisé avec prudence, mais il ne doit pas laisser penser que le droit est simplifié juridiquement.

À privilégier :

- clarification ;
- hiérarchisation ;
- structuration ;
- lisibilité.

### “Disruption”

À proscrire.

Le projet doit être moderne et ambitieux, mais il reste institutionnel, juridique et sensible.

---

## 11. Ce que le site ne doit pas évoquer

Le futur site ne doit pas ressembler à :

- un média sensationnaliste ;
- un site administratif froid ;
- un site juridique illisible ;
- un site trop académique ;
- un site pédagogique enfantin ;
- une plateforme startup ;
- une interface expérimentale trop risquée ;
- une vitrine graphique sans profondeur fonctionnelle.

L’interface doit rester sobre, stable, lisible et crédible.

---

## 12. Promesse de marque d’interface

La promesse de marque du futur site peut être formulée ainsi :

> **Un site qui rend le Conseil constitutionnel plus lisible, sans jamais affaiblir la valeur officielle de ses textes et décisions.**

Ou, de manière plus courte :

> **Comprendre l’institution. Lire les décisions. Approfondir avec confiance.**

---

## 13. Principes UI

La direction UI doit s’appuyer sur les principes suivants.

### Sobriété

Limiter les effets décoratifs inutiles.

### Lisibilité

Favoriser :

- les contrastes ;
- les tailles de texte adaptées ;
- les interlignes confortables ;
- les hiérarchies nettes ;
- les blocs courts ;
- les repères visuels utiles.

### Autorité

Utiliser :

- une composition stable ;
- une typographie maîtrisée ;
- des repères institutionnels ;
- une esthétique sérieuse.

### Modernité

Proposer :

- des composants actuels ;
- des interactions fluides ;
- une expérience responsive ;
- des parcours simples à utiliser.

### Accessibilité

Concevoir dès le départ pour une conformité élevée au RGAA.

### Écoconception

Limiter :

- les composants lourds ;
- les médias inutiles ;
- les animations décoratives ;
- les variations non nécessaires ;
- les charges de contenu difficilement maintenables.

### Modularité

Construire un design system réutilisable, maintenable et exploitable par le titulaire du lot 2.

---

## 14. Principes de contenu

Les contenus doivent être structurés selon une logique de lecture progressive.

Chaque contenu complexe doit pouvoir être abordé par étapes :

1. comprendre le sujet ;
2. identifier la décision ou le texte officiel ;
3. accéder aux références ;
4. approfondir avec les outils experts ;
5. continuer vers des contenus liés.

Les contenus d’aide à la compréhension doivent être :

- courts ;
- validés ;
- clairement distingués du texte officiel ;
- conçus comme des repères ;
- non substitutifs.

---

## 15. Badge de confiance éditoriale

Les pages comprenant des contenus d’aide à la compréhension doivent afficher un repère clair indiquant que seul le texte officiel fait foi.

### Formulation recommandée

> **Aide à la compréhension — seul le texte officiel fait foi.**

Cette formulation est préférable à “contenu pédagogique”, car elle est plus large, plus adulte et moins associée uniquement aux publics scolaires.

---

## 16. Relation avec “Découvrons notre Constitution”

Le site principal doit rester institutionnel.

Le site **“Découvrons notre Constitution”** conserve son rôle pédagogique autonome.

La marque du futur site ne doit donc pas chercher à reprendre une tonalité enfant, ado ou scolaire.

Elle doit plutôt orienter vers l’environnement pédagogique dédié lorsque le besoin correspond à ces publics.

L’hypothèse retenue pour le prototype d’avant-vente est de ne pas fusionner ce site, mais de créer des passerelles visibles et contextualisées.

Cette hypothèse devra être confirmée en phase de cadrage.

---

## 17. Définition du succès

La marque d’interface est réussie si elle permet de produire les perceptions suivantes.

### Côté citoyen

> **“Je comprends enfin.”**

### Côté expert

> **“Le site est plus moderne sans être simpliste.”**

### Côté institution

> **“Le site reste rigoureux, fiable et conforme à notre rôle.”**

### Côté titulaire du lot 2

> **“Les choix UX/UI sont clairs, documentés et réalisables.”**

---

## 18. Phrase de synthèse

> **Une interface républicaine, moderne et claire, qui rend les contenus du Conseil constitutionnel plus accessibles sans jamais affaiblir leur rigueur officielle.**
